Features
--------

Returns information about a radio frequency (US only for now)

Band Description, 
Band Type,
ITU Abbreviation,
ITU Number,
IEEE Band Name,
NATO Band Name,
Broadcast,
Amateur Radio Details (Type, Class, Max Power), - Not yet Implimented
Wavelength Range, - Not yet Implimented
Meters
